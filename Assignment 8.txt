  000002               *  FUNCTION: RETRIEVES INFORMATION ABOUT SEVERAL ORCHESTRAS   *                                              
  000003               *            WITH MUTUAL FUNDS ALONG WITH THE MUTUAL FUNDS    *                                              
  000004               *            THAT THEY USE AND TRANSACTION REQUESTS FOR THEM. *

  000029               *            CONTAINS: ORCHESTRA NAME, TRANSACTION TYPE FLAG, *                                              
  000030               *            ENDOWMENT CHANGE AMOUNT.                         *                                              
  000031               *                                                             *                                              
  000032               *  OUTPUT:   RPTFILE --                                       *                                              
  000033               *            OUTPUT TABLE CONTAINING INFORMATION ABOUT EACH   *                                              
  000034               *            TRANSACTION ALONG WITH TOTALS. ALSO CONTAINS A   *                                              
  000035               *            LIST OF ORCHESTRAS WITH ENDOWMENTS LESS THAN $1M.*

  000037               *            FUND NAME AND NUMBER,                            *                                              
  000038               *            INITIAL SHARE AND ENDOWMENT AMOUNT,              *

  000040               *            NEW SHARE / ENDOWMENT AMOUNT.                    *

  000042               *            # TRANSACTIONS PROCESSED,                        *                                              
  000043               *            INITIAL SHARES/ENDOWMENTS,                       *

  000045               *            LOW ENDOWMENT REPORT INCLUDES:                   *
